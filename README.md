I decided to draw a beach scene as my project 1. I love the beach and the ocean so I had a lot of inspiration to make this drawing. Initially, I wanted to draw some animals but all the details took very long and were complicated so I decided against it. The random function was useful for realistic placement of non-human things like whitecaps, grains of sand, and trees. The for loop was also very useful for repeating buildings and other manmade structures which needed to be in a uniform pattern. For the overall drawing, I drew inspiration from the beachside houses that I have seen on the east coast of the U.S. which are on stilts like this one. Surfing is popular in those east coast towns which is where I got the idea to have the surfboards in my drawing. As a whole I was proud of my drawing, and I enjoyed this project!
